Single Object Setup  
Just use the Cutscene - Simple Pan Cutscene macro, specifying the appropriate  
parameters:  
show actor - the object you want to show  
pan time - how long the pan should take  
pan ease - ease at the start (X) and end (Y) of the pan  
hold time - how long the shot should hold on the object after the pan is  
complete  
ﬁxed look from waypoint - optional: by default, we use the current game

Dialogue macro, specifying the speaker actor as an actor who has been made a  
speaker with the Dialogue - Speaker Create macro:
